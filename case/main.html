Mis on case vahendid

CASE vahendid võimaldavad automatiseerida, hallata ja lihtsustada süsteemiarenduse protsessi.
Näiteks vahendid lähtetingimuste analüüsiks, vooskeemi ja arendustöö ajakava koostamiseks,
dokumentatsiooni kirjutamiseks, programmiversioonide juhtimiseks, programmikoodi kirjutamiseks

Mille jaoks neid kasutatakse

CASE vahendeid kasutatakse, et automatiseerida tegevusi süsteemiarenduse elutsükli erinevates faasides

Kuidas jaotatakse case vahendeid

Süsteemiarenduse elutsüklit toetavad CASE vahendid jaotatakse kahte kategooriasse:
Uppercase
Lowercase

Milliseid vahendeid saab kasutada waterfall arendusmudeli elutsüklis

  • Projektijuhtimine:
    • Microsoft Project
    • Confluence
  • Nõuete kogumine ja analüüs:
    • Visio
    • Sparx Enterprise Architect
  • Arhitektuurse disaini väljatöötamine:
    • Visual Paradigm
    • Modelio
  • Programmikoodi loomine ja testimine:
    • Microsoft Visual Studio
    • JetBrains IDE-d
    • Selenium
    • Visual Studio Code
  • Toote konfiguratsioonihaldus:
    • Git
    • Subversion
